## Project Description

EduMind AI is a web-based academic assistant designed to help university students manage and improve their learning experience using artificial intelligence.

Students often face challenges such as:

- Poor study planning
- Lack of organized study materials
- Delays in obtaining academic guidance
- Difficulty tracking academic performance
- Manual quiz creation

EduMind AI provides a centralized and intelligent platform where students can receive personalized academic assistance. The system enables users to upload study materials, generate quizzes automatically, receive customized study plans, manage deadlines and tasks, and monitor their progress through dashboards.

The primary goal of EduMind AI is to improve academic performance through personalized and AI-driven learning support.

## Existing Systems / Research

### Google Classroom
Google Classroom helps manage assignments and communication but lacks personalized AI-based study planning and intelligent academic guidance.

### Quizlet
Quizlet provides study materials and quizzes but offers limited personalization and contextual academic support.

EduMind AI aims to combine multiple features into one intelligent academic ecosystem.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/edumind-ai.git
```

Move to the project directory:

```bash
cd edumind-ai
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```
